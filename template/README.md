# preset for vue cli 3

## How to use?

```
vue create --preset tonylua/vue-cli-3-preset-2019 your-project
```

## Commands

### Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test
```

### Compiles and hot-reloads for development
```
npm run serve
```
